# Café Order - カフェドリンク注文システム

シンプルなカフェドリンク注文Webアプリケーション。

## 機能

- **メニュー表示**: 10種類のドリンクをグリッド形式で表示
- **カート管理**: ドリンクの追加、数量変更、削除
- **注文処理**: 確認モーダル表示、注文番号の発行

## 使い方

1. `index.html` をブラウザで開く
2. メニューからドリンクをクリックしてカートに追加
3. 数量を調整（+/-ボタン）
4. 「注文する」ボタンをクリック
5. 確認画面で「確定する」をクリック
